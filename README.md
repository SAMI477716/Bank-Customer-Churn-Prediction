# Bank Customer Churn Analysis (Excel)

## Project Overview
This project involves a deep dive into banking data to identify patterns among customers who are likely to leave the bank. Using Microsoft Excel, I transformed raw data into a functional dashboard to provide actionable insights into customer retention.

## Key Analytics & Features

Data Cleaning: Standardized datasets, handled missing values, and prepared the data for analysis.

Pivot Tables & Charts: Created dynamic summaries of customer demographics (Credit Score, Age, Tenure, Balance).

Advanced Formulas: Utilized VLOOKUP, IF statements, and Aggregate functions for segmenting customer data.

Dashboarding: Built visual representations (Bar Charts, Pie Charts) to compare churn rates across different geographical locations and gender groups.
